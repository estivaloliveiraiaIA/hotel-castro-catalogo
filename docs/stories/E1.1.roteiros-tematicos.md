## Story
**As a** hóspede do Castro's Park Hotel,
**I want** acessar roteiros temáticos prontos (Jantar Romântico, Família, Negócios, etc.),
**so that** eu possa planejar minha estadia em Goiânia sem esforço, com sugestões curadas pelo hotel.

## Acceptance Criteria
1. Página `/itineraries` lista todos os roteiros disponíveis com card visual (ícone, título, duração, perfil)
2. Página `/itinerary/:id` exibe roteiro completo com paradas ordenadas, dicas do concierge e informações de cada lugar
3. Mínimo 3 roteiros temáticos publicados com dados reais
4. Navegação fluida entre home → roteiros → detalhe do roteiro → detalhe do lugar
5. Layout responsivo, mobile-first
