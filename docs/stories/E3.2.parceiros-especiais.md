## Story
**As a** hospede do Castro's Park Hotel,
**I want** identificar facilmente os parceiros do hotel e as vantagens exclusivas que eles oferecem,
**so that** eu possa aproveitar beneficios especiais durante minha estadia.

## Acceptance Criteria
1. Hook `usePartners` busca parceiros ativos do Supabase (join com places)
2. Badge "Parceiro do Hotel" visivel nos PlaceCards de lugares que sao parceiros
3. Badge "Parceiro do Hotel" visivel na pagina de detalhe do lugar (Place.tsx)
4. Secao "Parceiros do Hotel" na home page listando todos os parceiros ativos
5. Cada card de parceiro exibe: imagem do lugar, nome, categoria, badge_label e deal_description
6. Secao aparece apenas se houver pelo menos 1 parceiro ativo
7. Design do badge: dourado (hotel-gold), icone de estrela ou handshake, label configuravel via badge_label
8. Design consistente com o app premium
9. Responsivo — funciona em mobile

## Dev Notes

### Schema da tabela partners (ja existe no Supabase)
```sql
id               uuid primary key
place_id         text references places(id)
deal_description text    -- ex: "10% de desconto para hospedes do Castro's"
badge_label      text    -- ex: "Parceiro do Hotel" (configuravel pelo admin)
is_active        boolean default true
```

### Query Supabase (hook usePartners)
```ts
// Parceiros ativos com dados do lugar
supabase
  .from('partners')
  .select(`
    id,
    deal_description,
    badge_label,
    place:places(id, name, category, image, rating)
  `)
  .eq('is_active', true)
```

### Logica de badge no PlaceCard/Place
- usePartners retorna um Set<string> de place_ids parceiros
- PlaceCard recebe prop `isPartner?: boolean` e `partnerLabel?: string`
- Se isPartner=true, exibir <PartnerBadge label={partnerLabel} />

### PartnerBadge
- Cor: fundo hotel-gold/10, borda hotel-gold, texto hotel-gold
- Icone: estrela ou aperto de maos (Lucide: `Star` ou `Handshake`)
- Tamanho: pequeno para cards, medio para pagina de detalhe

### Secao na home
- Titulo: "Parceiros do Hotel"
- Subtitulo: "Beneficios exclusivos para hospedes"
- Cards horizontais (scroll em mobile)
- Mostrar todos os parceiros ativos (sem limite)
- Nao renderizar secao se partners.length === 0
